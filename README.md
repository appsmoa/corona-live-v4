# Corona Live

Live tracker for Coronavirus cases in Korea

[https://corona-live.com](https://corona-live.com)

## Tech Stack


- React
- Stitches (CSS in JS)
- SWR (Data fetching)
- D3.js (Data visualising)
- Zustand (State management)
- Vite (Frontend tooling)

## Setup

```
yarn && yarn start
```
